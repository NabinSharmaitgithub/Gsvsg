# **App Name**: ShadowText

## Core Features:

- Real-time Chat: Enable real-time, one-to-one anonymous chat using Socket.IO.
- Ephemeral Messages: Automatically delete messages immediately after they are viewed by the recipient.
- Secure Session: Identify users using temporary, expiring session IDs.
- Invite Link Generation: Generate a unique, one-time-use invite link for new chats.
- Typing Indicator: Show a typing indicator when the other user is typing a message.
- End-to-End Encryption: Secure all messages with client-side end-to-end encryption to protect user privacy.
- Disappearing Effect: Visually indicates messages are disappearing, similar to Snapchat's message expiration animation

## Style Guidelines:

- Primary color: Deep violet (#6A47BC) for a mysterious, modern feel.
- Background color: Very dark purple (#1A1526) for a dark scheme.
- Accent color: Electric indigo (#6352C4) to highlight user interface elements such as the send button and input fields.
- Body and headline font: 'Inter', a sans-serif for a neutral, objective style.
- Minimalist icons for chat actions (send, delete) in a monochromatic style.
- WhatsApp-style chat layout with a clean, mobile-first responsive design.
- Snapchat-style disappearing message animation to emphasize ephemeral nature.
